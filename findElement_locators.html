First matching element Many locators will match multiple elements on the page. The singular find element method will return a reference to the first element found within a given context. Evaluating entire DOM When the find element method is called on the driver instance, it returns a reference to the first element in the DOM that matches with the provided locator. This value can be stored and used for future element actions. In our example HTML above, there are two elements that have a class name of “tomatoes” so this method will return the element in the “vegetables” list.
